# Decisions

For new Architectural Decision Records (ADRs), please use the following template as a starting point:

-   [adr-template.md](adr-template.md) has all sections, with explanations about them.

The MADR documentation is available at [https://adr.github.io/madr/](https://adr.github.io/madr/) while general information about ADRs is available at [https://adr.github.io/](https://adr.github.io/).
